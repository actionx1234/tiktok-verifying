Mystery & True Crime Stories

Midnight Files is an independent creator channel covering real-world mysteries, unexplained disappearances, and unsolved cases through original short-form storytelling.

Every episode is independently researched and produced by the channel creator for audiences on TikTok and YouTube.

TikTok Integration

The Midnight Files TikTok app allows the channel creator to publish original video episodes directly to the connected TikTok account using the Content Posting API. Authentication is handled via TikTok Login Kit (OAuth 2.0). No automated posting occurs without the creator's explicit action.

No user data is collected, stored, or shared with any third party.
